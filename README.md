simpleEPG
=========

Dreambox and Vu+ boxes EPG for iOS7

This code compiles on Embarcadero Appmethod & Delphi XE5/XE6 (FireMonkey) and is primarily destined to run on iOS7 -
However it should run on Android devices too (not tested).
